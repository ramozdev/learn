# Table

## \<table\>: Table

Used to represent tabular data.

## \<tbody\>: Table Body

Encapsulates a set of table rows.

## \<tfoot\>: Table Foot

Encapsulates a set of table rows that represent the footer of the table.

## \<th\>: Table Header

## \<td\>: Table Data

## \<thead\>: Table Head

Encapsulates a set of table rows that represent the header of the table.

## \<tr\>: Table Row

Encapsulates a set/row of table cells.

## \<caption\>: Table Caption

```html copy
<table>
  <thead>
    <tr>
      <th>First Name</th>
      <th>Last Name</th>
    </tr>
```
