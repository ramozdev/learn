# Interactive elements

## \<details\>: Details disclosure

Used to create a disclosure widget from a `<summary>` element. The disclosure widget is a widget that can be expanded and collapsed by the user to reveal additional content.

```html copy
<details>
  <summary>Details</summary>
  <p>Details</p>
</details>
```

## \<summary\>: Disclosure Summary

Used as a summary, caption, or legend for a `<details>` element's disclosure box.

```html copy
<details>
  <summary>Summary</summary>
  <p>Details</p>
</details>
```
