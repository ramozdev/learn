## Main Root

### \<html\>: HTML Document / Root

The HTML element is the root element of an HTML document. All other elements must be descendants of this element.

```html copy
<html>
  <head>
    <title>Page Title</title>
  </head>
  <body>
    <h1>Page Heading</h1>
    <p>Page content</p>
  </body>
</html>
```
