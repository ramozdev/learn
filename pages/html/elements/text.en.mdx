# Text content

## \<blockquote\>: Block Quote

## \<dd\>: Description

## \<div\>: Content Division

Generic element used to group content.

```html copy
<div>
  <h1>I am a heading!</h1>
  <p>And I am a paragraph!</p>
</div>
```

## \<dt\>: Description Term

## \<figcaption\>: Figure Caption

## \<figure\>: Figure

## \<hr\>: Thematic Break (Horizontal Rule)

Creates a horizontal line.

```html copy
<p>Lorem ipsum dolor sit amet consectetur adipisicing elit. Minus, dolore!</p>

<hr />

<p>Lorem ipsum dolor sit amet consectetur adipisicing elit. Minus, dolore!</p>
```

## \<li\>: List Item

List element

```html copy
<ul>
  <li>Apples</li>
  <li>Bananas</li>
  <li>Watermelons</li>
</ul>
```

## \<menu\>: Menu

## \<ol\>: Ordered List

Ordered list.

```html copy
<ol>
  <li>Children</li>
  <li>Women</li>
  <li>Men</li>
</ol>
```

## \<p\>: Paragraph

Used to create paragraphs.

```html copy
<p>
  Lorem ipsum dolor, sit amet consectetur adipisicing elit. Aliquid reiciendis, iste laboriosam rem
  repellendus officia itaque quia placeat unde provident rerum magnam numquam porro totam.
</p>
```

## \<pre\>: Preformatted Text

Used to display text that preserves both spaces and line breaks.

```html copy
<pre>

/\  == \   /\  __ \   /\ "-./  \   /\  __ \   /\___  \   
\ \  __<   \ \  __ \  \ \ \-./\ \  \ \ \/\ \  \/_/  /__  
 \ \_\ \_\  \ \_\ \_\  \ \_\ \ \_\  \ \_____\   /\_____\ 
  \/_/ /_/   \/_/\/_/   \/_/  \/_/   \/_____/   \/_____/ 
</pre>
```

## \<ul\>: Unordered List

Used to create unordered lists.

```html copy
<ul>
  <li>Liberty</li>
  <li>Justice</li>
  <li>Privacy</li>
</ul>
```
