# Embedded content

## \<embed\>: Embed External Content

Used to embed external content, such as a video, an image, a page, or a plug-in.

```html copy
<embed
  src="https://www.youtube.com/embed/4gvSYEqOMn8"
  width="560"
  height="315"
  frameborder="0"
  allowfullscreen
/>
```

## \<iframe\>: Inline Frame

Used to embed external content, such as a video, an image, a page, or a plug-in.

```html copy
<iframe src="https://ramoz.dev" width="560" height="315" frameborder="0" allowfullscreen></iframe>
```
