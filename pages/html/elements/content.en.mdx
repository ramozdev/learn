# Content sectioning

## \<address\>: Contact Information

Used to provide contact information for the nearest `<article>{:html}` or `<body>{:html}` ancestor.

```html copy
<article>
  <h1>Lorem ipsum</h1>
  <p>Sit amet consectetur adipisicing elit. Distinctio animi provident delectus.</p>
  <address>
    <div>© 2023 Ramoz</div>
    <div>1234 Main St, Anytown, CA 12345</div>
    <div>
      <a href="mailto:example@gmail.com">
        <span class="visually-hidden">Email</span>
      </a>
      <a href="https://twitter.com/example">
        <span class="visually-hidden">Twitter</span>
      </a>
      <a href="https://facebook.com/example">
        <span class="visually-hidden">Facebook</span>
      </a>
    </div>
  </address>
</article>
```

## \<article\>: Article

Used to represent an independent piece of content. It could be a blog post, a news article, a forum post, a product card, etc.

```html copy
<article>
  <h1>Lorem ipsum</h1>
  <p>Sit amet consectetur adipisicing elit. Distinctio animi provident delectus.</p>
</article>
```

## \<aside\>: Sidebar

Used to provide additional information about the nearest `<article>{:html}` or `<body>{:html}` ancestor.

```html copy
<article>
  <h1>Lorem ipsum</h1>
  <p>Sit amet consectetur adipisicing elit. Distinctio animi provident delectus.</p>
  <aside>
    <h2>Related Posts</h2>
    <ul>
      <li><a href="#">Lorem ipsum</a></li>
      <li><a href="#">Sit amet</a></li>
      <li><a href="#">Consectetur adipisicing</a></li>
    </ul>
  </aside>
</article>
```

## \<footer\>: Footer

Contains links to resources, copyright information and/or contact information.

```html copy
<article>
  <h1>Lorem ipsum</h1>
  <p>Sit amet consectetur adipisicing elit. Distinctio animi provident delectus.</p>
  <footer>
    <div>© 2023 Ramoz</div>
  </footer>
</article>
```

## \<header\>: Header

Used to introduce some content or provide navigation links.

```html copy
<article>
  <header>
    <h1>A blog post</h1>
  </header>

  <p>
    Facilis blanditiis quibusdam aliquid quod possimus architecto, soluta magni, quaerat eveniet
    distinctio tenetur. Et, aperiam est?
  </p>
</article>
```

## \<h1\>-\<h6\>: Section Heading

Used for headings. `<h1>{:html}` is the highest section heading level, `<h6>{:html}` is the lowest.

```html copy
<h1>h1 heading</h1>
<h2>h2 heading</h2>
<h3>h3 heading</h3>
<h4>h4 heading</h4>
<h5>h5 heading</h5>
<h6>h6 heading</h6>
```

## \<main\>: Main Content

Used for the main content inside the `<body>{:html}`. It could contain the body of a blog or the main functionality of an app.

```html copy
<h1>Lorem ipsum</h1>
<main>
  <p>
    Lorem ipsum dolor, sit amet consectetur adipisicing elit. Distinctio animi provident delectus
    similique, velit eligendi!
  </p>
</main>
```

## \<nav\>: Navigation Section

Used to provide navigation. Common examples include sidebar menus, breadcrumbs and top navigation bar.

```html copy
<nav>
  <ul>
    <li><a href="#">lorem</a></li>
    <li><a href="#">ipsum</a></li>
    <li>consectetur adipisicing</li>
  </ul>
</nav>
```

## \<section\>: Section

Used to group related content. It could be a blog post, a news article, a forum post, a product card, etc.

```html copy
<section>
  <h1>Lorem ipsum</h1>
  <p>Sit amet consectetur adipisicing elit. Distinctio animi provident delectus.</p>
</section>
```
