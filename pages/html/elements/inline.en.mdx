# Inline text semantics

## \<a\>: Anchor

Used to create a hyperlink to another page or to an element within the same page.

```html copy
<a href="https://www.ramoz.dev">ramoz.dev</a>
```

## \<abbr\>: Abbreviation

## \<br\>: Line Break

## \<cite\>: Citation

## \<code\>: Inline Code

Used to indicate that a section of text is a computer code.

```html copy
<code>console.log('Hello World!')</code>
```

## \<data\>: Data

## \<dfn\>: Definition

## \<em\>: Emphasis

## \<kbd\>: Keyboard Input

## \<mark\>: Marked Text

## \<q\>: Short Inline Quotation

## \<samp\>: Sample Output

## \<small\>: Small Print

## \<span\>: Generic Inline Container

## \<strong\>: Strong Importance

## \<sub\>: Subscript

## \<sup\>: Superscript

## \<time\>: Time

## \<var\>: Variable

## \<wbr\>: Word Break Opportunity
