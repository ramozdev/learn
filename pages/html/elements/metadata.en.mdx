# Document Metadata

## \<head\>: Document Metadata (Header)

Used to contain metadata (data about data) for the document, including links to or definitions of scripts and style sheets.

```html copy
<!DOCTYPE html>
<html>
  <head>
    <title>Page Title</title>
  </head>
  <body>
    <h1>My First Heading</h1>
    <p>My first paragraph.</p>
  </body>
  <script>
    console.log('Hello World')
  </script>
</html>
```

## \<link\>: External Resource Link

Used to link to external resources such as style sheets.

```html copy
<link rel="stylesheet" href="mystyle.css" />
```

## \<meta\>: Metadata

Used to specify metadata about an HTML document. Metadata will not be displayed on the page, but will be machine parsable.

```html copy
<meta charset="UTF-8" />
```

This is not how pages are styled in modern web development. You will learn how to style pages using TailwindCSS for a better developer experience.

## \<title\>: Document Title

Used to specify a title for the document. This is shown in the browser's title bar or on the page's tab.

```html copy
<title>Cool Website - Nice Dev</title>
```
