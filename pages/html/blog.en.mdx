# Blog

In this section you will learn about HTML commonly used in blogs.

## Boilerplate

Start by creating a new file called `index.html` and add the following boilerplate code:

```html copy
<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8" />
    <meta name="viewport" content="width=device-width, initial-scale=1.0" />
    <meta name="description" content="Pablo Pérez's blog." />
    <title>Learn About HTML - Pablo Pérez</title>
  </head>

  <body></body>
</html>
```

Now start adding content inside the `<body>{:html}`.

## Navigation

First add a navigation bar. This is a list of links that will allow users to navigate between pages.

```html copy
<nav>
  <ul>
    <li><a href="#">Home</a></li>
    <li><a href="#">About</a></li>
    <li><a href="#">Blog</a></li>
    <li><a href="#">Contact</a></li>
  </ul>
</nav>
```

## Main content

Inside `<main>{:html}` we have an `<article>{:html}` with the following content:

### Header

`<header>{:html}` is used to wrap the title and the date of the article.

```html copy
<header>
  <h1>HTML</h1>
  <time datetime="2023-03-03">03.03.2023</time>
</header>
```

### Figure

`<figure>{:html}` is used to wrap the audio player and the caption.

```html copy
<figure>
  <figcaption>Listen to this blog post</figcaption>
  <audio controls src="/media/html.mp3"></audio>
</figure>
```

### Section

`<section>{:html}` is used to separate the content of the article.

```html copy
<section>
  <h2>What is HTML?</h2>

  <p>
    <dfn><abbr title="HyperText Markup Language">HTML</abbr></dfn> is the standard markup language
    for creating web pages. With HTML you can create your own
    <a href="https://en.wikipedia.org/wiki/Website" target="_blank" rel="noopener noreferrer"
      >website</a
    >.
  </p>

  <p>
    Lorem ipsum <em>dolor</em> sit amet consectetur adipisicing elit. Suscipit laudantium ipsa,
    nihil, ducimus corrupti saepe aperiam error iusto, esse doloribus
    <strong><em>culpa non similique!</em></strong> Totam, eveniet. Reprehenderit qui autem officia
    accusantium.
  </p>
</section>
```

### Aside

`<aside>{:html}` is used to wrap the content that is not directly related to the main content of the article.

```html copy
<aside>
  <p>
    <mark>You only need HTML to create a website</mark>, but nowadays, almost
    <span>no website</span> is created using only HTML.
  </p>
</aside>
```
