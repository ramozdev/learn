# Node.js

Node.js es un entorno de tiempo de ejecución JavaScript de back-end que ejecuta código JavaScript fuera de un navegador web.

## Verifica tu versión de Node.js

Abre una terminal y ejecuta el siguiente comando para ver la versión de Node.js instalada en tu computadora:

```bash copy
node -v
```

Si no tiene instalado Node.js, verás un mensaje de error similar al siguiente:

```bash
zsh: command not found: node
```

Si tienes instalado Node.js, verás la versión, por ejemplo:

```bash
v18.14.0
```

Debes tener instalada la versión LTS de Node.js. Compruebe cuál es la versión LTS de Node.js actualmente en [nodejs.org/en/download](nodejs.org/en/download).

## Instalación

Ve a [snapcraft.io/node](https://snapcraft.io/node) (Debian/Ubuntu) e instala Node.js.

Si está utilizando una distribución de Linux diferente, ve a [nodejs.org/en/download/package-manager](https://nodejs.org/en/download/package-manager/) y descarga el instalador apropiado para tu sistema operativo.

## Verifica instalación

Ejecuta el siguiente comando para ver la versión de Node.js instalada en tu computadora:

```bash copy
node -v
```

Cuando instalas Node.js, también se instala Node Package Manager (npm). Puedes verificar que npm esté instalado ejecutando el siguiente comando:

```bash copy
npm -v
```

Si tienes npm instalado, verás la versión, por ejemplo:

```bash
9.2.0
```
