# Git

Git es un sistema de control de versiones que te permite realizar un seguimiento de los cambios en los archivos y colaborar con otros desarrolladores.

## Instalación

Abre una terminal y ejecuta el siguiente comando para ver la versión de git instalada en tu sistema:

```bash copy
git -v
```

Si no tienes `git` instalado, instálalo usando el siguiente comando:

```bash copy
sudo apt-get install git
```

Una vez finalizada la instalación, abre una nueva terminal y verifica que se haya instalado `git`.

Si no puedes instalar `git` siguiendo estos pasos, consulta la [documentación de git] (https://git-scm.com/download/linux).

## GitHub

Crea una cuenta en [GitHub](https://github.com/signup) para almacenar su código en la nube.

## Configuración

Cada vez que hace un commit en git, debes proporcionar tu nombre y dirección de correo electrónico. Esto es para que git pueda rastrear quién hizo el commit. Puedes configurar esto globalmente en tu computadora.

Completa y ejecuta los siguientes comandos con el mismo nombre para mostrar y correo electrónico principal que configuró en su cuenta de github.

```bash copy
git config --global user.name “[nombre apellido]”
```

```bash copy
git config --global user.email “[correo electrónico válido]”
```

Ahora has configurado git en su computadora. Aprenderás a usar git en otra sección.
