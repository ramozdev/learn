# Node.js

## Check Your Node.js Version

Open a terminal. Follow these steps to open a terminal in your operating system. Press `windows + r` to open Run. Type `cmd` and press enter.

Type the following command and press enter.

```bash
node -v
```

If you get a node version in this format: `vx.x.x`, where `x` is a number, then you already have Node.js installed. Otherwise, you need to install Node.js

## Installation

Go to [nodejs.org/en/download](https://nodejs.org/en/download/) and download the installer for your operating system.

Run the installer and follow the installationg guide.

## Check

To check if the installation was succesful open a terminal and type the following commands:

```bash
node -v
```

You should get your node version as `vx.x.x`, where `x` is a number.

```bash
npm -v
```

You should get your npm version as `vx.x.x`, where `x` is a number.
