# Required Software

You will need to install and learn to use the following software.

- Terminal/Command Line Interface: An integrated software in your operating system.
- Git: Distributed source control system
- Node.js: A back-end JavaScript runtime
- Visual Studio Code: Text editor

Now select your operating system and follow the instructions to install these programs and learn the basics of each one.

- [Linux](/setup/linux/cli)
- [MacOS](/setup/macos/cli)
- [Windows](/setup/windows/cli)
